# food-delivery
# Food Delivery Web Application

A full-stack Food Delivery Web Application built using the MERN stack.  
The application allows users to browse food items, add them to cart, and place orders, while an admin panel is available to manage food products.

---

##  Features

- User Authentication (Login & Registration)
- Browse Food Items
- Add to Cart functionality
- Place Orders
- Admin Dashboard for managing food items
- REST API integration
- Responsive UI

---

## Tech Stack

**Frontend**
- React.js
- CSS
- Axios
- React Router

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB
